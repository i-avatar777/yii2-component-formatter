# yii2-component-formatter

Компонент для форматирования расширяющий стандартный датами в русском календаре

## Как использовать

Пропиши в конфигурационном файле:
```php
return [
    // ...
    'components' => [
        'formatter'       => [
            'class' => '\iAvatar777\components\FormatterRus\FormatterRus',
            // ...
        ],
    ],
];
```
